# Auto-Matrix
油猴/控制台.js脚本. 
一键完成中山大学Matrix编程平台(matrix.sysu.edu.cn)作业.

**注意：Matrix编程平台是提升编程技能的重要练习平台。知识的巩固需要反复实践。**

# 使用方法

[控制台]auto-matrix.js
1. 复制代码内容
2. 浏览器打开Matrix平台进入第一个题目中, 按f12打开开发者模式, 选择控制台.
3. 粘贴到控制台
4. 在弹出框内填写API信息.
5. 启动!

[油猴]auto-matrix.js
1. 复制代码内容
2. 在浏览器篡改猴插件内添加新脚本->粘贴.
3. 浏览器打开Matrix平台进入第一个题目中, 弹出窗口, 在弹出框内填写API信息.
4. 启动!
