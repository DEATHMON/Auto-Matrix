# Auto-Matrix
油猴/控制台.js脚本. 快速完成中山大学Matrix编程平台(matrix.sysu.edu.cn)作业.

**Matrix编程平台是提升技能的重要练习平台。请牢记：知识的巩固需要反复实践。因此，在使用本脚本之前，请务必确认自己已经掌握了相关基础知识。**

# 使用方法

[控制台]auto-matrix.js
1. 复制代码内容
2. 浏览器打开Matrix平台进入第一个题目中, 按f12打开开发者模式, 选择控制台.
3. 粘贴到控制台
4. 在弹出框内填写API信息.
5. 启动!

[油猴]auto-matrix.js
1. 复制代码内容
2. 在浏览器篡改猴插件内添加新脚本->粘贴.
3. 浏览器打开Matrix平台进入第一个题目中, 弹出窗口, 在弹出框内填写API信息.
4. 启动!
